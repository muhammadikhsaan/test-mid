## How To Run
  ### PREPARE
  - create database
  - run `docker-compose up -d` to running database via docker
  ### BACKEND
  - open terminal
  - run `npm install`
  - run `npm run start:dev`
  - table database automatically create (TypeORM Migration)
    ### API DOCUMENTATION
    - swagger ui -> http://localhost:3030/documentation
  ### FRONEND
  - open terminal
  - run `npm install`
  - run `npm start`
